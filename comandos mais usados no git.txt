git init = para iniciar a pasta .git.
git clone = para clonar a pasta de alguém.
git status = verificar de houve mudanças nos arquivos.
git add . = para subir todos os arquivos para a pasta e depois o commit.
git add nome_do_arquivo.extensão = para subir um arquivo especifico para a pasta e depois o commit.
git commit -m "titulo do que você fez ou alterou" = salvar no git para depois o push
git branch = mostra a branch que estamos trabalhando.
git log = mostra os commits feitos.
git reflog = mostra os commits resumidos com as iniciais dos numero de referencia.
git log --pretty=oneline = mostra os commits resumidos com os numero de referencia completos.
git diff = mostra todas as operações feitas no arquivos.
